# Contributing to POWER8 Projects

POWER8 Projects - Ubuntu 22.04 build, PSE LLM, Darwin cross-compile.

## Development Setup

```bash
git clone https://github.com/MingYu5/power8-projects.git
cd power8-projects
```

## Supported Systems

- IBM POWER8 (ppc64le)
- Ubuntu 22.04

## Contributing

1. Fork the repo
2. Create a feature branch
3. Test on real POWER8 hardware
4. Open PR against `Scottcjn/power8-projects:main`

Questions? Open an issue!
